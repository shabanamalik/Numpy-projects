# 📊 Store Sales Analyzer

A beginner-level data analysis project built 
using NumPy to analyze monthly sales data 
of a store with 3 products over 12 months.

## 🎯 What This Project Does
- Calculates annual sales per product
- Finds the best performing month
- Tracks cumulative (running) sales
- Filters high sales months (above 200K)
- Calculates sales after 10% tax deduction
- Generates a quarterly sales report

## 🛠️ Concepts & Functions Used
- `np.random` — generating fake sales data
- `np.sum` — annual and monthly totals
- `np.argmax` — finding best month
- `np.cumsum` — running sales total
- `np.where` — filtering high sales
- `np.reshape` — quarterly grouping
- Broadcasting — applying tax on all values

## ▶️ How to Run
pip install numpy
python sales_analyzer.py
